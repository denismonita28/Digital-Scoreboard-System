# Digital-Scoreboard-System

• Designed and implemented a finite state machine (FSM)-based digital scoreboard for two teams, enabling score
increment and decrement using push-button inputs.

• Developed real-time score visualization on a 4-digit multiplexed 7-segment display, including clock division, button
debounce logic and display decoding modules.

• Achieved a stable and user-controlled scoring system capable of accurate real-time updates and reliable operation in
hardware digital environments.
